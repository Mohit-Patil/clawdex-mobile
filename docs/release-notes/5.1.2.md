# Release Notes 5.1.2

## Store / TestFlight Notes

- Added the local preview browser with improved desktop and overview browsing modes.
- Made transcript tool activity easier to inspect, including grouped tool calls and visible changed filenames.
- Smoothed drawer, composer, and navigation interactions across the app.
- Added app-wide font preferences and refined several core screens and sheets.
- Improved reconnect and bridge restart reliability for day-to-day use.

## Detailed Notes

### Browser and Preview

- Added the mobile local preview browser flow.
- Reworked browser desktop mode into a more stable overview shell.
- Expanded browser controls and improved preview address display, session handling, and refresh behavior.

### Transcript and Tool Activity

- Improved chat transcript controls and composer stability.
- Added grouped tool-call expansion with per-tool output inspection.
- File-change timeline items now identify the changed file instead of only showing a generic success label.
- Rendered conversation compaction as a dedicated separator and improved activity/loading indicators.

### Navigation and UI Polish

- Refined drawer drag behavior, reopen timing, reconnect behavior, and browser-screen swipe handling.
- Reduced composer visual height, improved file mention handling, and tightened main screen rerender behavior.
- Added app-wide font preferences and refined popup sheet sizing, layout, and header affordances.

### Reliability

- Hardened bridge restart cleanup.
- Fixed CI, formatting, lint, and review issues that blocked release quality.
